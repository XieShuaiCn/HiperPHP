模板标签说明

网站变量
<!--{web.path}-->网站根网址
<!--{web.title}-->网站标题

模板变量
<!--{template.名称}-->嵌套指定名称的模板

全局变量
标签结构<!--{global.变量名}-->
使用说明：
在模板中调用全局变量，目前只支持变量名，不支持数据名、对象名等。

高级标签说明

循环标签
标签结构<!--[list={数据:参数1,参数2,参数3,参数4}]--><!--%row.数据属性%--><!--[/list]-->
使用说明：
1、数据：可以为模型名等。

2、标签中间可以写模板代码，模板代码中使用“row.数据属性”来调用该数据的某个属性值，模板代码中可以包含基本的模板变量。

3、该模板标签必须闭合。

代码标签
标签结构<!--[code={php}]--><!--[/code]-->
使用说明：
在标签中间写入PHP代码，便可执行。便签中不要使用echo等输出语句，输出使用：$code_ret.='你要输出的内容';
此标签会受网站系统影响，且安全性差，慎用！！！

模板解析顺序
模板变量——网站变量——全局变量——古画变量——文献变量——高级标签
